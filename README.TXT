Se nos pide que se lea un archivo CSV por medio de readFile_CSV. Esa lectura hay que guardarla en una lista por el método dataPalabras.
La clase data recibe la lista y la itera (por medio del método iterList); la manda a una clase "compPalindromo". Esta clase va iterando a lo largo de una cadena de texto para comprobar si la primera letra coincide con la última y va avanzando a través de la palabra. 
UI recibe y detecta si las palabras analizadas (por medio de un true false). El método "mostrarResultado" muestra a las palabras que sí son palindromas. 
El método cerrar cierra el programa.
